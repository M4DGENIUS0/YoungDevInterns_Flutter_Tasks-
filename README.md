## Installation

Install my-project with npm

```bash
  - Clone this repository:
  $ git clone https://github.com/M4DGENIUS0/YoungDevInterns_Flutter_Tasks-
  
  - Enter in directory:
  $ cd YoungDevInterns_Flutter_Tasks--main
  
  - For install dependencies:
  $ flutter packages get
  
  - Run the app: 
  $ flutter run
```
